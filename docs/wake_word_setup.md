# 语音唤醒方案说明

## 当前方案：VAD 人声唤醒（纯 Dart）

本项目使用**纯 Dart 实现的 VAD（语音活动检测）**方案，无需任何注册、无需联网、完全免费开源。

### 工作原理

```
[麦克风] → 音频能量分析 → 检测到人声 → 触发唤醒
```

- 通过 `record` 包采集音频能量
- 分析 RMS 振幅判断是否有人声活动
- 连续检测到足够多次 → 触发唤醒
- 完全离线运行，不发送任何数据到网络

### 灵敏度调节

在 `wake_word_service.dart` 中可调参数：

```dart
threshold: 0.06    // 振幅阈值 (0~1), 越小越灵敏
requiredHits: 3    // 需要连续检测到多少次才触发, 越大越不易误触
```

调用 `setSensitivity(0.0~1.0)` 动态调节灵敏度。

### 与 Porcupine 对比

| 特性 | VAD 方案 (当前) | Porcupine (已废弃) |
|------|----------------|-------------------|
| 注册 | ❌ 不需要 | ❌ 需要公司邮箱 |
| 网络 | ✅ 完全离线 | ✅ 完全离线 |
| 费用 | ✅ 完全免费 | ⚠️ 有使用限制 |
| 唤醒方式 | 任何人声唤醒 | 特定关键词"小猫小猫" |
| 精确度 | 任何人说话都会唤醒 | 只说"小猫小猫"才唤醒 |

> 代码框架已保留扩展性。
